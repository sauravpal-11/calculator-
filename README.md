🧮 Python GUI Calculator

A simple and clean calculator application built using Python's Tkinter library.
It supports basic arithmetic operations with a user-friendly graphical interface.

---

🚀 Features

- ➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division
- 📊 Percentage calculation (%)
- 🧹 Clear button (C) to reset input
- ⌫ Backspace to delete last digit
- 🖥 Real-time display updates
- 🎨 Styled buttons and clean UI
- 🔢 Continuous calculation (result reused)

---

🛠 Technologies Used

- Python 3
- Tkinter (GUI library)
